Aquarium-to-Neo4j Driver
========================

This code provides a driver for transferring entries from an
Aquarium session to a Neo4j graph database.

.. code-block:: console

    echo "Install using pip"
    pip install aqneodriver

    echo "Using the command line api..."
    aqneodriver --help
