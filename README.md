# AquariumToNeo4jETL

(alpha)

Adapter to convert Aquarium DB to Neo4j DB using optimized http queries to an Aquarium instance.

The Neo4j DB can be used for data exploration, graph algorithms, or anything else the Neo4j DB is used for.
